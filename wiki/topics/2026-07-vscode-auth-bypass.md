# Visual Studio Code 미인증 권한상향 취약점

Microsoft Visual Studio Code 소프트웨어 개발 도구에서 미인증 공격자가 네트워크를 통해 권한을 상향할 수 있는 결함. 전 세계 수백만 개발자의 필수 도구 침해로 공급망 공격이 우려된다.
