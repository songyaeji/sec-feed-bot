해커들이 Visual Studio 프로젝트 파일을 변조해 다단계 백도어를 배포하는 공격이 발견되었다. C++/C# 개발 환경을 표적으로 하며, 백도어 외에도 정보탈취기, 클립보드 낚아채기, 암호화폐 채굴기, 파일 감염자를 함께 배포한다. Doctor Web 연구원이 2025년 4분기부터 관찰했으며 현재도 활발하다.
