**n8n** 워크플로우 자동화 플랫폼(1.123.61, 2.27.4, 2.28.1 이전)의 레거시 MySQL v1 노드에서 SQL 인젝션. executeQuery 작업이 `{{ ... }}` 표현식을 **파라미터화 없이** 원본 SQL 문자열에 직접 삽입. 워크플로우가 사용자 입력을 변수로 받으면 공격자가 **SQL 인젝션** 가능. 자동화 플랫폼 데이터베이스 침해 경로.
