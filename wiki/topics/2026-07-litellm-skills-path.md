**LiteLLM** 1.83.7-stable 이전 버전의 Skills 기능이 업로드된 ZIP 아카이브에서 파일 경로를 충분히 검증하지 않아, 인증된 사용자가 LiteLLM API 경로 접근 권한이나 해당 키가 있으면 경로 순회로 임의 파일을 추출할 수 있는 취약점.
