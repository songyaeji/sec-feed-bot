# React Router 오픈 리디렉션 취약점

**React Router** (@remix-run/router v1.23.2 이전, react-router v7.0.0~v7.11.0)에서 Framework Mode, Data Mode, unstable RSC modes에서 loaders 또는 actions 발생 오픈 네비게이션 리디렉션이 안전하지 않은 URL로 이동할 수 있는 취약점. **CVSS 8.0**.
